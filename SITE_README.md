DH101 — Local site

This is a lightweight local site wrapper for the course materials already in this repository.

Quick start

1. Open `index.html` in your browser (double-click or right-click → Open with).
2. Use the top navigation to jump to the `Training` section or open the markdown pages linked under Quick Links.

Notes

- The site links directly to the markdown files under `pages/`, `makes/`, and `reflections/` so your browser will show raw markdown unless you serve the site through a static site generator.
- Theme toggle is persisted in `localStorage`.

Want a converter? Ask me to convert the markdown pages into HTML pages or to scaffold a simple static site generator build step.
