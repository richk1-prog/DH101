How I Use AI

For me and AI use I tried to keep only when I couldn't understand how to do something, but had an idea I really wanted to see done. I would explain each prompt and the intent, along with what I kept from each prompt

Example 1: I asked for a light/ dark mode on my website, but I decided to get rid of the function when I had a better idea for how the main pages should look.

Example 2: I asked the following to get the matrix flowing text behind the pages: "change the backdrop not the bubbles for text, or buttons just the background okay. Have it become black with evershifting 1s and 0s at this hex code #77FF00."
